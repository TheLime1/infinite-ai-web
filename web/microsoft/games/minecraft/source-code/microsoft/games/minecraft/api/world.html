Minecraft World API

This section details the Minecraft World API within the source code.

Sub-Modules

  • Block API: Information about interacting with blocks.
  • Entity API: Information about interacting with entities.
  • Chunk API: Information about interacting with chunks.
  • Dimension API: Information about interacting with dimensions.

Core Functionality

The World API provides access to:

  • Getting and setting block data.
  • Spawning and manipulating entities.
  • Accessing chunk data.
  • Managing dimensions.
  • Handling world events.
