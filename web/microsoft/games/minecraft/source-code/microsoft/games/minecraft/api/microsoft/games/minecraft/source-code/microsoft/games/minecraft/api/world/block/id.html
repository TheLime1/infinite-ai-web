Minecraft Source Code: Block ID API

This section details the API for retrieving block IDs within the Minecraft source code.

API Details

This API allows developers to programmatically access and manage block IDs within a Minecraft world.

Methods

  • getBlockId(x, y, z): int - Returns the block ID at the specified coordinates.
  • setBlockId(x, y, z, id: int): void - Sets the block ID at the specified coordinates.
  • getBlockName(id: int): string - Returns the name of the block associated with the ID.
