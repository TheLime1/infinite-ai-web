Python Data Types

Python has several built-in data types, each serving different purposes.

Common Data Types

  • Integers (int): Whole numbers, like 10, -5, 0.
  • Floating-point numbers (float): Numbers with a decimal point, like 3.14, -2.5.
  • Strings (str): Sequences of characters, like "Hello", 'Python'.
  • Booleans (bool): Represent truth values, either True or False.
  • Lists (list): Ordered, mutable sequences of items, like [1, 2, "apple"]. Learn more at Lists.
  • Tuples (tuple): Ordered, immutable sequences of items, like (1, 2, "apple").
  • Dictionaries (dict): Key-value pairs, like {"name": "Alice", "age": 30}. Learn more at Dictionaries.
  • Sets (set): Unordered collections of unique items, like {1, 2, 3}.

Understanding data types is crucial for writing effective Python code. See Variables for information on how to store these values.
