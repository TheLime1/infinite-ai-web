Python Lists

Lists are a fundamental data structure in Python. They are ordered, mutable collections that can hold items of different data types.

Creating Lists

Lists are created using square brackets []:

my_list = [1, 2, 3, 'a', 'b', 'c']

Accessing List Elements

You can access elements in a list using their index, starting from 0:

first_element = my_list[0]  # first_element will be 1

List Methods

Python lists have many useful methods, such as:

  • append(): Adds an element to the end of the list.
  • insert(): Inserts an element at a specific index.
  • remove(): Removes the first occurrence of a value.
  • pop(): Removes and returns the element at a specific index.
  • len(): Returns the length of the list.

List Comprehensions

List comprehensions provide a concise way to create lists.

squares = [x**2 for x in range(10)] # creates a list of the squares of numbers from 0 to 9
